二、系统设计与架构
本系统整体分为四个逻辑层级：前端采集层、融合网关层、通信传输层、后
端AI平台层。
3

2.1 前端采集层
在船舶甲板、制高点或关键作业区部署一台 5G 超轻量智能布控球。该设备
具备防水特性，适合海上高盐雾环境，支持1080P高清视频采集及360°连续旋
转。
2.2 融合网关层（核心节点）
在船长室或机柜内部署一台N2计费网关。智能布控球通过网线接入N2网关
的 LAN 口（1~8口支持 PoE 供电或直接数据传输）。船载一体化动中通卫星终
端及 MESH自组网通信设备均通过网线接入 N2网关的通信端口。N2网关作为船
端的网络控制枢纽，统筹管理下联设备的数据流向。
2.3 通信传输层（智能路由调度）
优先链路（4G/5G）：当船舶行驶在近海或港口等有运营商信号的区域时，
系统优先调用4G/5G网络通道（通过布控球内置的5G模块）回传视频流。
备用链路（卫星动中通）：当船舶驶向远洋，失去 4G/5G 信号覆盖时，N2
网关及系统路由策略自动将流量切换至WAN口连接的“船载动中通卫星终端”。
该天线具备全自动寻星和波束切换功能，即使在恶劣海况下也能保障链路的稳定。
协同共享链路（MESH 自组网）：在多艘作业船只处于自组网信号覆盖范围
内时自动建立 MESH 编队网络。
 近海场景：若编队中某一艘或多艘船只接收到5G信号，组网内的其他船
只可利用 MESH 网络路由，跨船共享使用5G高速通道。
 远洋场景：所有船只依赖卫星信道时，若某一艘船的卫星通信发生遮挡
或中断，N2网关及路由策略将自动把该船的数据流量通过 MESH 网络引
流至组网内其他通信正常的船只，借用其卫星信道完成数据回传。
2.4 后端 AI 平台层（岸端）
岸端管理中心接收到通过公网或卫星专网传输过来的实时视频流。岸边服务
器集群运行 AI 视觉算法，针对船舶场景进行定制化分析（如：船员未穿戴救生
衣/安全帽检测、甲板异常闯入、火灾烟雾识别等），并在指挥大屏上进行预警

弹窗。

3.2 5G 超轻量智能布控球
产品简介：
5G 超轻量智能布控球是基于移动监控、应急布控、车载监控等需求设计开
发的一体化产品，由高清晰摄像机、阵列红外照明灯组、高精度云台和减震吸盘
等模块组成。设备全铝机身，内置4G/5G传输模块、蓝牙、大容量锂电池等，满
足视频实时传输、录像下载回放、人脸识别、车牌识别、消噪拾音等功能需求，
可应用于多种应急演练、安保布控场景。

 支持双向语音对讲（需配备蓝牙耳机或手咪）；

 内置 Wifi 模块，可使用手机、平板直接预览视频，手机、平板也可通过球

3.4 MESH 自组网终端
产品简介：
该终端采用先进的软件无线电架构（SDR），集成无中心多跳高性能自组网
技术，无需中心节点即可实现设备间直接通信，支持动态路由与中继转发，具备
卓越的语音、数据、视频及位置信息多业务承载能力；在复杂地形下可快速部署
现场网络，实现车与车、车与无人机、车与单兵之间的不间断通信，其强大的抗
干扰能力与动态组网特性充分满足高机动性、隐蔽性和高带宽需求，在抢险救灾、
应急保障、常态巡逻、单兵组网、车辆组网及无人机集群自组网等场景中展现出
卓越性能，确保通信畅通与任务高效执行。
产品特性：
 灵活组网：支持无中心自组成网，载波带宽灵活可配，可构建点对多点、链
状中继、网状、混合等多种网络拓扑，适应复杂多变的通信需求。
 全IP互联：采用全IP化设计，支持各类数据透传，易于与异构通信系统互
联互通，促进多网业务融合交互，实现更广泛的通信覆盖。
 抗干扰性：选配宽频载波跳频技术，显著提升抗干扰能力；智能选频工作模
式可实时避开干扰频点，确保网络稳定；结合ARQ差错控制传输机制，有效
降低数据丢失率，保障数据传输的可靠性。
 高可靠性：创新采用功放与核心板一体化设计，具备出色的抗振防护性能，
确保设备在恶劣环境下稳定运行。
 无人机专用波形（可选）：可选专为无人机应用场景定制开发的专用波形，
优化传输时延，并确保飞控数据优先传输，提升无人机控制效率。
